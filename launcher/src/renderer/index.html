Gachaboard — first-time setup

A shared whiteboard for audio, video, design files, and more

Sign in with Discord. For remote collaborators, install Tailscale and share the URL.

The EXE alone does not start the server. Choose the folder that contains the full project (e.g. unpacked ZIP). If you move the folder, set it again in Settings (gear).

Create a Discord application:

  1. At the Discord Developer Portal, click New Application
  2. Under OAuth2 → General, copy Client ID and Client Secret
  3. Under OAuth2 → Redirects, add:
    • http://localhost:18580/api/auth/callback/discord
    • https://<your-tailscale-host>.ts.net/api/auth/callback/discord (when using Tailscale; hostname is visible after startup)
