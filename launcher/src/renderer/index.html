設定・取説

Gachaboardとは

自分のPCをサーバーにして動かすホワイトボードです。音楽・映像・デザインなど、あらゆるファイルを自由に貼り付けて共有できます。データはすべて自分のPC内に保存されます。

ログイン方法

Discordアカウントを使用してログインします。初回のみ、Discordのデベロッパーポータルでアプリを作成し、以下のURLを「Redirects」に登録してください。

http://localhost:18580/api/auth/callback/discord

Tailscaleで起動すると表示されます

外部から参加してもらうには

離れた場所にいる人と使う場合は、Tailscaleを利用します。参加者にもTailscaleをインストールしてもらい、起動後に表示されるURLを共有してください。同じネットワーク内であれば、localhostのURLを共有するだけで利用可能です。

※サーバー主は、Tailscale管理画面で「MagicDNS」と「HTTPS」を有効にする必要があります。

招待コード

プロジェクトごとに参加メンバーを管理できるよう、招待コード方式（ワークフロー・ワークボード）を採用しています。

Gachaboard 初回設定

音楽・映像・デザインファイルを貼り付けて、みんなで一緒に使えるホワイトボード

Discordアカウントでログインします。外部から参加してもらう場合はTailscaleをインストールし、URLを共有してください。

Discordでのアプリ作成手順:

  1. Discord Developer Portal で「New Application」を作成
  2. 「OAuth2」→「General」で Client ID と Client Secret を取得
  3. 「OAuth2」→「Redirects」に以下を追加:
    • http://localhost:18580/api/auth/callback/discord
    • https://<あなたのTailscaleホスト>.ts.net/api/auth/callback/discord（Tailscale 利用時。ホスト名は起動後に確認可能）
