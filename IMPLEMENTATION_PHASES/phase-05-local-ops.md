# Phase 5: ローカル運用と永続化

## 目的

- ローカル1TBストレージへデータを集約し、再起動しても保持される運用を確立する。

## 実装対象

- `docker-compose.yml`
- `nextjs-web/.env.example`
- `nextjs-web/README.md`（運用手順）

## タスク

1. Postgresデータの永続ボリュームを固定。
2. オブジェクト保存先（ローカルディスク）を固定。
3. バックアップ/削除手順をGUI導線前提で記載。
4. Dドライブ前提の起動手順を明文化。

## 依存

- 単独着手可（ただし `Phase 2` と整合を取る）

## 完了条件

- 再起動後もデータが残ることを確認し、保存先パスがドキュメント化されている。
