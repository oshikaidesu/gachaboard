# Hooks ディレクトリ構成

責務ごとにサブフォルダで整理しています。

| フォルダ | 内容 |
|----------|------|
| `yjs/` | Yjs 同期・永続化・Awareness・YMap 購読 |
| `invite/` | 招待リンク・招待情報・参加 |
| `workspace/` | ワークスペース一覧・詳細 |
| `board/` | ボードスナップショット・復元・同期状態 |
| `media/` | 波形・OGP・アセット状態・メディアプレイヤーコメント |
| （ルート） | ファイルドロップ・矢印削除・プレビュー・リアクション等の汎用フック |
